# Bitcoin Index Time Series Analysis

Statistical time series analysis of the Bitcoin price index — decomposing price behaviour, identifying structural patterns, and evaluating forecasting models on a highly volatile financial asset.

---

## Approach

- Time series decomposition: trend, seasonality, and residual components
- Stationarity testing (ADF/KPSS) and differencing
- ARIMA/GARCH model identification, fitting, and diagnostic checking
- Volatility analysis: assessing heteroskedasticity in returns
- Forecast evaluation: in-sample and out-of-sample performance

---

## Tools

`R` · `forecast` · `tseries` · `ggplot2`
