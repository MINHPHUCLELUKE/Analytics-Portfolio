# Climate-Responsive Time Series Forecasting

Time series forecasting that incorporates climate variables as predictors — going beyond standard univariate models to capture how temperature, rainfall, or other environmental factors drive trends in the target series.

---

## Approach

- Exploratory analysis of climate covariate relationships with the target series
- Baseline univariate forecasting (ARIMA/ETS) for comparison
- Dynamic regression and/or ARIMAX: incorporating climate covariates as external regressors
- Seasonal decomposition accounting for climate-driven seasonality
- Forecast accuracy comparison: RMSE, MAE, MAPE across model types

---

## Tools

`R` · `forecast` · `fable` · `ggplot2`
